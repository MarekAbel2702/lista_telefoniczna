ListaTelefoniczna

Prosta aplikacja Android napisana w Javie do zarządzania listą kontaktów.
Umożliwia dodawanie osoby (imię, nazwisko, numer telefonu, email) oraz wyświetlanie ich na liście.

Funkcjonalności

dodawanie nowego kontaktu

wyświetlanie listy kontaktów

formularz z danymi użytkownika

nawigacja między fragmentami

RecyclerView z adapterem

Jak uruchomić projekt

Otwórz projekt w Android Studio

Poczekaj aż Gradle się zsynchronizuje

Wybierz emulator lub podłącz telefon

Kliknij Run ▶

Wymagania

Android Studio

Android SDK

Emulator lub urządzenie z Androidem
